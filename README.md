CO2 Emission Analysis and Vehicle Clustering
This project provides a comprehensive analysis of CO2 emissions based on various vehicle characteristics, such as engine size and cylinder capacity. By leveraging statistical modeling and machine learning, the project explores the relationship between vehicle specifications and their environmental impact.

Key Features
Predictive Modeling: Implementation of multiple regression techniques, including Linear Regression and Non-linear Regression, to estimate emission levels.

Data Visualization: Interactive and static visualizations to identify trends, correlations, and outliers in the emission dataset.

Vehicle Clustering: Utilization of unsupervised learning to group vehicles with similar performance and emission profiles.

Exploratory Data Analysis (EDA): Deep dive into the factors most heavily influencing carbon footprints in the automotive industry.

Tech Stack
Language: Python

Libraries: pandas, matplotlib, seaborn, scikit-learn
